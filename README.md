# WhatsApp Unhack Tool

This tool helps you regain control of your WhatsApp account if it has been compromised or hacked. It simulates the process of unhacking your account with a loading bar and provides a success message upon completion.

## Features

- Simulates the unhacking process with a loading bar.
- Provides a success message upon completion.

## Requirements

- Python 3.x

## Installation

1. Clone the repository:
   bash
   git clone https://github.com/coedextech/whatsapp_unhack_tool.git
   cd whatsapp_unhack_tool
   pip install -r requirements.txt
   python main.py
